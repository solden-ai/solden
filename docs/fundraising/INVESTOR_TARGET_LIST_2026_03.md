# Solden Investor Target List

Date: 2026-03-29

This is a best-fit outreach list for Solden's current round:

- raise: $2M-$2.5M
- instrument: post-money SAFE
- target cap: $15M
- stage: pre-seed with real product, 3 design partners, and 1 committed pilot starting in May
- story: finance execution layer for finance teams; first skill is Gmail-native AP

This is not a random VC list. It is filtered for investors that are plausibly active now for:

- pre-seed or seed
- fintech and/or finance infrastructure
- enterprise workflow / vertical software
- AI-native application software
- Europe and US venture ecosystems

Scoring:

- Tier A: highest-probability targets to start with now
- Tier B: strong next wave once the first meetings are underway
- Tier C: plausible but more dependent on warm intros, process heat, or stronger pilot proof

Important:

- "Highly probable" here means best fit based on public stage/thesis positioning as of 2026-03-29. It is still an inference, not a guarantee.
- Partner selection matters more than firm logo. For each fund, find the partner most likely to care about fintech, enterprise workflow, or AI-native vertical software.
- Use warm intros wherever possible. Cold outreach should be reserved for funds that clearly invite direct inbound.

## Messaging Notes

Lead with this version of the story:

- not "AI accounting"
- not "AP automation"
- not "generic agent platform"
- yes: "AI execution layer for finance teams"
- yes: "first skill is Gmail-native AP"
- yes: "removes approval chasing and duplicate ERP entry"

Best angles by investor type:

- fintech specialists: finance execution infrastructure, control layer between inbox, approvals, and ERP
- software / SaaS funds: AI-native workflow software for finance operators
- AI-native funds: narrow, high-frequency vertical workflow with clear human-in-the-loop guardrails
- Europe-to-US funds: European / cross-border software wedge with obvious US expansion path
